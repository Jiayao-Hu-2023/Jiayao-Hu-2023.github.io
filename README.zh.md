## 1. 项目概述

Cyber-Lotto 是一个部署在 GitHub Pages 上的静态单页应用（SPA）。它采用赛博朋克视觉风格，模拟中国福利彩票"双色球"玩法。

**核心宗旨**：这是一个纯粹的运气测试游戏，不涉及任何金钱交易，不设充值入口，无用户登录系统。

## 2. 核心规则与逻辑

### 2.1 资产系统
- **初始化**：每次网页加载或刷新，用户资产重置为 **20 点**。
- **持久化**：不使用 LocalStorage 或数据库，刷新即重置。
- **消耗**：每注扣除 2 点。
- **奖励**：根据中奖规则增加资产点数（单位：点）：
  - 一等奖：+5,000,000
  - 二等奖：+1,000,000
  - 三等奖：+3,000
  - 四等奖：+200
  - 五等奖：+10
  - 六等奖：+5

### 2.2 投注逻辑
- **投注模式**：
  - **单式投注**：红球选6个，蓝球选1个。
  - **复式投注**：红球可选 7-20 个，蓝球可选 2-16 个。系统自动计算生成的注数。
- **选号交互**：
  - **手动选号**：点击数字选中/取消。
  - **随机选号**：点击按钮，系统随机生成一注完整号码（支持复式参数）。
  - **无限机选**：用户可连续点击随机选号，直到满意为止。
- **倍数功能**：
  - 默认 1 倍，最大 99 倍。
  - **费用公式**：`总点数 = 注数 × 2 × 倍数`。
  - **奖金公式**：`总奖金 = 单注奖金 × 中奖注数 × 倍数`。
- **订单确认流程**：
  1. 用户选号完毕，点击"生成订单"。
  2. 系统计算总注数与总花费。
  3. **资产校验**：若 `总花费 > 当前资产`，弹出赛博风格警告框（Glitch效果），阻止生成订单。
  4. 确认无误后，扣除资产，进入摇奖环节。

### 2.3 摇奖逻辑
- **触发机制**：投注成功后，页面自动切换至摇奖界面。
- **随机性**：结果完全随机，不受用户选择影响。
- **流程控制**：
  - 界面显示两台摇奖机器（左侧红球机，右侧蓝球机）。
  - **阶段一（红球）**：红球机启动，每 **10秒** 摇出 1 个红球，共摇出 6 个。
  - **阶段二（蓝球）**：红球摇完后，蓝球机启动，摇出 1 个蓝球。
  - **总耗时**：约 70 秒。
- **动画细节**：
  - 球在机器内有物理碰撞感或赛博空间的漂浮感。
  - 球被摇出时，通过管道滑入展示区，数字初始模糊逐渐清晰。
  - 进度条配合倒计时显示。

## 3. UI/UX 设计规范

### 3.1 视觉风格
- **主题**：赛博朋克 / 霓虹科技
- **配色**：
  - 背景：深黑 `#050505` 或暗蓝 `#0a0e14`
  - 红球：霓虹红 `#ff2a6d`，选中/摇出时带发光效果
  - 蓝球：霓虹青 `#05d9e8`，选中/摇出时带发光效果
  - 文字：白色/灰白，重点信息用高亮黄或亮绿

### 3.2 页面布局

**状态 A：选号大厅**
- **顶部**：Logo + 资产显示（大号字体）
- **中部**：
  - 左区：红球选号池（33个球，网格布局）
  - 右区：蓝球选号池（16个球，网格布局）
  - 底部浮动栏：已选号码预览 + 倍数选择器 + [清空] [机选] [确认订单] 按钮
- **弹窗**：订单确认弹窗，资产不足警告弹窗

**状态 B：摇奖现场**
- **顶部**：当前期号 + "正在摇奖中..." 状态文字
- **中部**：
  - 双摇奖机动画区域
  - 已摇出号码展示区（按顺序排列）
- **底部**：进度提示（如"红球第 3/6 个，距离下一个球还有 8s"）

**状态 C：结果结算**
- 摇奖结束后自动弹出
- **对比展示**：用户号码 vs 开奖号码，命中号码高亮闪烁
- **结算信息**：中奖等级、中奖注数、奖金总额
- **操作**：[返回选号] 按钮

## 4. 技术实现要求

- **技术栈**：HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript (ES6+)
- **状态管理**：使用简单的 JS 对象管理 `asset`, `selectedBalls`, `winningBalls`, `gameState`
- **动画实现**：
  - CSS Keyframes 处理球的发光、抖动效果
  - JavaScript `setInterval` 控制摇奖的时间轴逻辑
- **数学计算**：
  - 组合数公式 C(n, m) 用于计算复式注数
  - 兑奖逻辑遍历对比算法

## 6. 快速开始

1. **克隆仓库**：
   ```bash
   git clone https://github.com/your-username/Cyber-Lotto.git
   cd Cyber-Lotto
   ```

2. **在浏览器中打开**：
   - 直接打开 `index.html`
   - 或使用本地服务器获得更好体验：
     ```bash
     # 使用 Python
     python -m http.server 8000
     
     # 使用 Node.js
     npx serve .
     ```

3. **访问地址**：
   - 本地：`http://localhost:8000`
   - GitHub Pages：`https://your-username.github.io/Cyber-Lotto/`
